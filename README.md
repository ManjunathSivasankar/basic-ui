# Basic UI - Reusable Components

This project demonstrates simple reusable UI components built using React.js and basic HTML/CSS. The goal of this project is to create clean and reusable components that can be used across different parts of a web application.

## Features
- Simple reusable components
- Clean and minimal UI
- Beginner-friendly React project
- Organized component structure

## Technologies Used
- React.js
- HTML5
- CSS3
- JavaScript (ES6)
